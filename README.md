# Murat Ergüzel - Portfolyo Sitesi

Bu proje, Murat Ergüzel'in kişisel portfolyo sitesidir. Anasayfada AMP teknolojisi kullanılarak geliştirilmiştir.

## Özellikler

- AMP teknolojisi ile hızlı yükleme
- Responsive tasarım
- Portfolyo projeleri
- İş deneyimi ve eğitim bilgileri
- Sosyal medya bağlantıları
- İletişim bilgileri

## Kurulum

1. Projeyi klonlayın
2. `index.html` dosyasını bir web tarayıcısında açın

## Geliştirme

Anasayfa AMP kurallarına uygun olarak geliştirilmiştir. Değişiklik yaparken AMP validasyonunu kontrol etmeyi unutmayın.
